# Руководство по подключению интеграций

## 1. Smart Bridge API (отправка в КГД)

### Как получить доступ

Smart Bridge — это шлюз «электронного правительства» для отправки форм налоговой отчётности.

**Шаг 1: Регистрация на портале Smart Bridge**
1. Перейдите на https://sb.egov.kz
2. Нажмите «Регистрация»
3. Заполните данные организации (БИН, наименование)
4. Загрузите ЭЦП для идентификации

**Шаг 2: Получение API-ключа**
1. Войдите в личный кабинет Smart Bridge
2. Перейдите в раздел «API»
3. Создайте новое приложение
4. Получите `client_id` и `client_secret`

**Шаг 3: Подключение к сервису КГД**
1. В разделе «Подписки» найдите «Приём форм налоговой отчётности»
2. Нажмите «Подключить»
3. Подпишите заявку ЭЦП
4. Дождитесь одобрения (1-3 рабочих дня)

**Формат подключения:**
```javascript
const SMART_BRIDGE = {
  auth_url: 'https://sb.egov.kz/oauth/token',
  api_url: 'https://sb.egov.kz/api/v1',
  client_id: 'YOUR_CLIENT_ID',
  client_secret: 'YOUR_CLIENT_SECRET'
}
```

## 2. Kaspi Business API (автоимпорт выписок)

### Как получить доступ

Kaspi предоставляет API для партнёров (бизнес-аккаунт required).

**Шаг 1: Регистрация бизнес-аккаунта**
1. Убедитесь, что у вас есть **Kaspi Pay** или **Kaspi Магазин**
2. Перейдите на https://business.kaspi.kz
3. Зарегистрируйтесь как партнёр

**Шаг 2: Подача заявки на API**
1. Перейдите в раздел «Для разработчиков»
2. Заполните заявку на доступ к API:
   - Название компании
   - БИН
   - Описание использования
   - Контактные данные
3. Отправьте заявку

**Шаг 3: Получение OAuth-доступа**
После одобрения вы получите:
- `client_id`
- `client_secret`  
- `redirect_uri`

**Примечание:** Kaspi API доступен только для верифицированных бизнес-партнёров. Срок рассмотрения заявки: 5-14 рабочих дней.

**Альтернатива: Выписки через файл**
Пока ждёте доступ к API, используйте экспорт выписок в CSV/Excel из мобильного приложения Kaspi:
1. Откройте приложение Kaspi
2. Перейдите в «Мой Банк»
3. Выберите счёт → «Выписка»
4. Укажите период и формат «CSV»
5. Загрузите файл в приложение

## 3. Push-уведомления и календарь

### Реализация в браузере

Push-уведомления работают через **Service Worker** и **Push API**.

**Что нужно сделать:**

1. **HTTPS-сервер** (обязательно)
   - GitHub Pages поддерживает HTTPS ✓
   - Service Worker работает только на HTTPS

2. **Разрешение пользователя**
   ```javascript
   async function requestNotificationPermission() {
     if ('Notification' in window) {
       const permission = await Notification.requestPermission()
       return permission === 'granted'
     }
     return false
   }
   ```

3. **Напоминания о сроках**
   - Срок подачи 910 за 1 п/г: до 15 августа
   - Срок подачи 910 за 2 п/г: до 15 февраля
   - Отправлять уведомление за 7 дней и за 1 день

### Автономные уведомления (без сервера)

**Вариант 1: Браузерные напоминания**
Приложение проверяет сроки при каждом открытии и показывает напоминание.

**Вариант 2: Календарь (iOS/Android)**
Генерация .ics файла для импорта в календарь:
```javascript
function generateCalendarFile(deadlines) {
  const ics = `BEGIN:VCALENDAR
VERSION:2.0
PRODID:-//Declaration910//NONSGML v1.0//EN
${deadlines.map(d => `BEGIN:VEVENT
DTSTART:${d.date.replace(/-/g, '')}
SUMMARY:${d.title}
DESCRIPTION:${d.description}
END:VEVENT`).join('\n')}
END:VCALENDAR`
  downloadFile('deadlines.ics', ics)
}
```

## 4. eGov.kz — проверка по БИН

### Бесплатное API (ограниченное)

**data.egov.kz** предоставляет открытые данные:

```javascript
// Поиск организации по БИН
async function getOrgByBIN(bin) {
  const response = await fetch(
    `https://data.egov.kz/api/v4/ds/binary?bin=${bin}`
  )
  return response.json()
}
```

**Примечание:** Это API не требует авторизации, но имеет ограничения по количеству запросов.

### Полный доступ к eGov

Для получения полной информации (выписки, справки) требуется:
1. Авторизация через ЭЦП или SMS
2. Подписка на государственные услуги
